Термины:
Состояние — некоторые данные, которые хранит объект.
Поведение — действия, которые может совершать объект.
Сигнатура метода — имя метода плюс параметры (причем порядок параметров имеет значение).
сериализация (Serialize) - из объекта в файл
Deserialize - из файла в объект

Принципы ООП
Абстракция
1. Наследование
- выносить общую логику наверх
2. Инкапсуляция
- сокрытие реализации (скрывает от нас сложную логику, которая не имеет для нас значение)
- защита состояния (позволяет защитить данные, если нужно их нельзя отредактировать)
3. Полиморфизм (интерфейсы - полиморфизм поведения, когда одно и тоже действие может быть реализовано по-разному.
Полиморфизм типов - когда можно работать одинаково с разными типами данных.)
